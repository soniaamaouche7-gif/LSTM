# Electricity Consumption Prediction

This project predicts electricity consumption using a cleaned household power consumption dataset.

## Dataset
- `household_power_consumption_clean.csv` : cleaned dataset after preprocessing.
- Features: `hour`, `day`, `month`, `prev_consumption`
- Target: `consumption`

## Notebook
- `notebook_prediction.ipynb` : contains LSTM model for prediction, training, evaluation, and plots.
- Data pre-processing and cleaning.
